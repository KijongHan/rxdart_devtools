title: rxdart_devtools
description: A DevTools extension for inspecting RxDart streams in your Flutter app.
---

`rxdart_devtools` is a Dart DevTools extension that surfaces your `rxdart` streams as a panel inside Flutter DevTools — see active subscriptions, recent events, and stream metadata while your app is running.

:::caution
This project is currently in early development. APIs and the panel UI are still scaffolding.
:::
